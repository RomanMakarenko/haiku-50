# Haiku 50

**Генератор хайку через OpenAI API**

Мінімалістичний веб-застосунок для створення японських трирядкових віршів — хайку. Користувач вводить 3–7 ключових слів, обирає мову, регулює рівень «гостроти» (50 Васабі) та отримує унікальну хайку, згенеровану ШІ.

## Особливості

- 🌶 **50 Васабі (Spiciness Level)** — регулювання рівня гостроти від 0 (класика) до 6 (абсурд)
- 🌐 **12 мов генерації** — українська, англійська, японська, китайська, арабська та інші
- 📜 **Японська естетика** — Bento Grid, каліграфічний пензель, сувій, бонсай на тлі
- 💾 **Історія** — останні 100 хайку зберігаються в браузері (localStorage)
- 🛡 **Безпека контенту** — трирівневий захист від нецензурного або агресивного контенту
- 📱 **Адаптивний дизайн** — коректно відображається на мобільних пристроях

## Встановлення та запуск

### Локальна розробка

1. Клонувати репозиторій
2. Встановити залежності:
   ```bash
   npm install
   ```
3. Створити файл `.env` у корені проєкту:
   ```
   OPENAI_API_KEY=sk-your-key-here
   PORT=3000
   ```
4. Запустити сервер:
   ```bash
   node api/server.js
   ```
5. Відкрити в браузері: `http://localhost:3000`

### Тестування

```bash
npm test              # одноразовий запуск
npm run test:watch    # watch-режим під час розробки
```

## API

### POST /api/generate

Генерує хайку на основі ключових слів.

**Request body:**
```json
{
  "keywords": "сакура, дощ, тиша",
  "language": "uk",
  "spiciness": 3
}
```

**Response (200):**
```json
{
  "haiku": "рядок 1\nрядок 2\nрядок 3",
  "fallback": false
}
```

### GET /api/health

Перевірка стану сервера. Повертає 200.
